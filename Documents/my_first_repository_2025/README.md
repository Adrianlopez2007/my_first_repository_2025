# Portafolio académico de Adrián

Este sitio web recoge información sobre mí, mi grado en Ingeniería Informática
y los contenidos trabajados en la asignatura Fundamentos de la Ingeniería
Informática.

## Descripción del trabajo

El proyecto consiste en un sitio web estático desarrollado con HTML y CSS.
Las páginas se organizan en una página principal (`index.html`) y varias
secciones dentro de la carpeta `public` (about, contacts, net, topic, degree y fii).

La estructura del proyecto es:

- `index.html`: página principal del sitio.
- `public/about.html`: información personal y currículum.
- `public/contacts.html`: formulario de contacto.
- `public/net.html`: enlaces a las páginas de compañeros.
- `public/topic.html`: tema sobre ciberseguridad básica.
- `public/degree.html`: descripción del grado y asignaturas.
- `public/fii.html`: página dedicada a Fundamentos de la Ingeniería Informática.
- `css/style.css`: hoja de estilos común.
- `images/`: carpeta con las imágenes utilizadas.

## Problemas durante el desarrollo

- Aprender a usar Git y GitHub (configurar el repositorio, hacer commits y
  subir cambios con `git push`).
- Organizar correctamente la estructura de carpetas (`css`, `images`, `public`)
  y las rutas relativas en los enlaces y en la hoja de estilos.
- Crear todas las páginas HTML manteniendo un mismo diseño y menú de navegación.

## Conclusiones

Este trabajo me ha ayudado a repasar HTML y CSS, a trabajar con un proyecto web
con varias páginas y a familiarizarme con Git (add, commit,
push) y la publicación con GitHub Pages.
